        View Source Trento.Support.JsonbSerializer (Trento Web v2.2.0)

        Serialize to/from PostgreSQL's native jsonb format. Requires events to be defined by the defevent macro.

        Configuration example:

        config :trento, Trento.EventStore,
  serializer: Trento.Support.JsonbSerializer,
  column_data_type: "jsonb",
  types: EventStore.PostgresTypes

        Summary

        Functions

        deserialize(term, config)

        Deserialize a jsonb value from the event store into a struct. This function is called by the event store when reading events and snapshots from the database. In case the type supports upcasting (i.e. it has a upcast/2 function), the event is wrapped in an IntermediateEvent struct. The upcaster protocol is implemented for IntermediateEvent and it will take care of calling the new!/1 function after the upcasting is done.

        serialize(term)

        Callback implementation for EventStore.Serializer.serialize/1.
