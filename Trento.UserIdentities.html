        create_user(user_identity_params, user_params, user_id_params)

        View Source

        redefining the PowAssent create user method, this is called when the user login through idp and a user identity does not exists on our database.

        If a user with the same username exists on our database, the user will be recovered and associated with the idp identity, otherwise the user will be created.
