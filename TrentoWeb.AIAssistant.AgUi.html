    TrentoWeb.AIAssistant.AgUi (Trento Web v3.1.2)

    Copy Markdown View Source

    AG-UI protocol emission helpers for TrentoWeb.AIAssistantChannel.

    Every public function takes and returns the socket so the channel pipes cleanly. Wire encoding (camelCase + nil-strip + JSON round-trip) delegates to AgUi.Encoder.EventEncoder.encode_json/1.
